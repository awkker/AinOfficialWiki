## 材料准备

- 发票

### 1. 报名费

- 一般在比赛平台开具发票。
- 发票背面左上角使用**铅笔**写报名费，张三，xx元。
- 发票正面使用黑色中性笔签字。
- 如果发票包含张三李四王五三人的报名费，需要签三个人的名字。
- 支付凭证使用黑色中性笔签字。
- 如果支付凭证包含张三李四王五三人，需要签三个人的名字，并注明xxx、xxx等三人报名费。

### 2. 路费

- 从车票、机票购买处开发票。如果有电子行程单可选，并且平台注明等效为发票，也可以开电子行程单。
- 发票背面左上角使用**铅笔**写火车票/机票/打车费，从<出发地>→<目的地>，HH点MM分出发/登机，HH点MM点到达，张三（全部乘车人），xx元
- 发票正面使用黑色中性笔签字。
- 如果发票包含张三李四王五三人的路费，需要签三个人的名字。
- 支付凭证使用黑色中性笔签字。
- 如果支付凭证包含张三李四王五三人，需要签三个人的名字，并注明xxx、xxx等三人火车票/机票/打车费，从<出发地>→<目的地>，HH点MM分出发/登机，HH点MM点到达，张三（全部乘车人）。
- 如果购买机票，需要证明机票价格更便宜或者相差不多。
- 如果打车需要上高速，无法直接凑滴滴开高速费的发票；需要联系师傅要付款凭证，使用上面的条形码，在“山东省高速公路人工收费电子发票”小程序开具发票。

### 3. 住宿费

- 一般从酒店前台处开发票。
- 发票背面左上角使用**铅笔**写住宿费，从MM月DD日→MM月DD日，共几晚，张三（全部乘车人），xx元
- 发票正面使用黑色中性笔签字。
- 如果发票包含张三李四王五三人的住宿费，需要签三个人的名字。如果其中一个人不在队伍，需要单独注明xxx在xx队伍，以及两个队伍分别的额度占比。
- 酒店报销不超过350/晚
- 支付凭证使用黑色中性笔签字。
- 如果支付凭证包含张三李四王五三人，需要签三个人的名字，并注明从MM月DD日→MM月DD日，共几晚，张三（全部乘车人），xx元，张三（全部乘车人）。
- 如果有额度分配情况，也需要在支付凭证写上说明。

### 4. 虚拟产品费用（服务器租赁、API调用等）

适用于服务器租赁、云服务、模型/API 调用、对象存储、域名解析等虚拟产品费用报销。

#### 需要准备的材料

- 发票
- 付款凭证
- 消费记录
- 平台账号说明页（**单独一页**）

#### 平台账号说明页

需单独打印一页，写明：

- 平台名称
- 平台账号
- 账号使用人姓名
- 使用用途
- 使用时间范围
- 对应消费金额
- 相关消费记录说明

可参考如下格式：

> 平台名称：xxx
>  平台账号：xxx
>  使用人姓名：张三
>  使用用途：比赛/项目所需服务器租赁、API 调用等
>  使用时间：20xx年xx月xx日—20xx年xx月xx日
>  消费金额：xx元
>  使用情况与记录：用于xxx，详见后附消费记录、付款凭证、发票。

#### 发票要求

- 需由对应平台开具发票。
- 发票背面左上角使用**铅笔**写：
   **平台账号：xxx，张三，xx费用，xx元**
- 这里的“xx费用”应写明具体类别，如：
  - 服务器租赁费
  - API 调用费
  - 云存储费
  - 域名/解析服务费
- 发票正面使用黑色中性笔签字。
- 如一张发票涉及多人共同使用，需写清各自姓名及费用分摊情况。

#### 付款凭证要求

- 需提供支付成功截图、支付订单页、银行卡/支付宝/微信支付记录等可证明付款完成的材料。
- 付款凭证使用黑色中性笔签字。
- 并在凭证空白处注明：
   **平台账号：xxx，张三，xx费用，xx元**
- 如涉及多人共同使用，需注明各自姓名、费用类别及金额分配。

#### 消费记录要求

- 需提供平台后台导出的账单、消费明细、订单记录、调用记录、资源使用记录等。
- 消费记录上需注明：
   **平台账号：xxx，张三，xx费用，xx元**
- 消费记录应尽量体现：
  - 平台账号
  - 消费时间
  - 消费项目
  - 消费金额
  - 使用情况或调用情况

#### 说明

- 虚拟产品费用原则上需做到：**账号明确、用途明确、消费可追溯、付款可核验、发票可对应**。
- 发票、付款凭证、消费记录三者的金额和时间范围应尽量保持一致。
- 如因平台原因无法在单据原件上手写说明，可另附情况说明，并确保能对应到平台账号、姓名和费用。
